# 🏠 Sistema de Agendamento de Aluguel de Imóveis

Projeto desenvolvido em PHP + MySQL para gerenciamento de imóveis e agendamentos de locação.

---

## 📌 Funcionalidades

- 🔐 Login e cadastro de usuários
- 🏠 Cadastro de imóveis
- 📅 Agendamento de reservas
- 📋 Listagem de agendamentos
- ✏️ Edição de agendamentos
- 🗑 Exclusão de registros
- 🔗 Relacionamento entre usuários e imóveis
- 📊 Dashboard com estatísticas

---

## 🧰 Tecnologias utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- XAMPP

---

## 🗄️ Banco de Dados

Nome: `sistema_agendamento`

Tabelas:
- usuarios
- imoveis
- agendamentos

---

## ⚙️ Como executar o projeto

1. Instale o XAMPP
2. Coloque o projeto em:C:\xampp\htdocs\
3. Inicie Apache e MySQL
4. Importe o banco de dados no phpMyAdmin
5. Acesse:http://localhost/SistemaAgendamento/
http://sistemaagendamento.nfy.fyi/cadastro.php
http://sistemaagendamento.nfy.fyi/login.php
